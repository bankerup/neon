About Neon

Neon is a file hosting and sharing script written by Bankerup.me. You can use neon to create your own file hosting and sharing website. currently Is provides the following features:

  • Multiple files uploading
  • Chunked file uploading
  • Searching for files
  • User registration and login
  • Responsive design
  • Simple Administrator panel
  • Simple file manager
  • Simple profile editor

Neon is a single page web application. The client side of Neon is written in Javascript, CSS, and HTML. The server side of Neon is written in Javascript (NodeJS).
